FORMULARIO

Elementos Básicos

Para empezar un formulario, primero necesitas la etiqueta "form", aquí dentro es donde pondrás todas las demás etiquetas necesarias.

Para la mayoría de cosas, primero pondrás la etiqueta "label", que es el nombre de lo que debe poner la persona (por ejemplo, Nombre), y después la etiqueta "input", que es el dato a poner, variará según lo que quieras poner, mediante el atributo "type=". Si quieres que tenga que rellenar un campo si o si antes de enviar, en el input pon el atributo "required".

Para hacer el botón de enviar, simplemente haz un "input" con el tipo "submit". Si también deseas un botón de restablecer, que borra todo lo que se ha puesto, es igual que el de enviar, solo que el tipo es "reset".

Si quieres que haya un texto de fondo, como el que se ve de "escriba aquí...", en la etiqueta "input" pon el atributo "placeholder=".









Patrón de Contraseña

Para que la contraseña tenga requisitos, usa el atributo "pattern={6,8}", en este caso, el primer número es el mínimo de longitud y el segundo el máximo, pon los números que quieras. También puedes poner mayúsculas con A-Z y minúscuals con a-z. Ten en cuenta que los corchetes sirven para cuando quieres que contenga algo, y la llave para el número de caracteres.

Subir Archivos

Para añadir una opción que permita subir archivos, simplemente pon el tipo "file" en la etiqueta input


Seleccionar Color

Para que la persona pueda seleccionar un color, pon el tipo "color" en la etiqueta input


Menú Desplegable

Primero ponemos el label, después dos etiquetas nuevas. Primero "select", que es la etiqueta donde pondremos todas las opciones. Segundo "option", que contendrá el nombre de cada una de las opciones.


Área de Texto

Para un espacio donde la persona pueda escribir lo que quiera, tienes la etiqueta "textarea" (acompañado antes de su "label", si así lo quieres). Dentro de la etiqueta puedes poner un texto que vendrá ya puesto.
