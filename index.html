Estructura Básica

Un archivo HTML tiene varias etiquetas siempre presentes, son las siguientes:

  • Etiqueta html. Dentro de esta etiqueta se encuentra todo el contenido del archivo.
  • Etiqueta head. En esta etiqueta se encuentran cosas como estilos dentro del HTML (aunque se deben evitar), metadatos, título de la pestaña, menú y enlace con hoja de estilo y javascript (algunas de estas cosas están explicadas más adelante).
  • Etiqueta body. Esta etiqueta tiene dentro todo el contenido del archivo HTML, la cara visible.
  • Etiqueta footer. Esta etiqueta a veces se omite, se trata de el pié de página.

Metadatos

Como mencionamos antes, los metadatos suelen estar en el "head" y son datos relativos al archivo html, que se pueden modificar, se ponen con la etiqueta "meta" y lo que varían son los atributos. Algunos más usados son:

  • charset="UTF-8". Este atributo hace que el archivo sea compatiblo con Ñ y tildes. Aunque se pueden especificar otro formato de caracteres cambiando el "UTF-8".
  • name="". En este metadato se pueden poner diferentes cosas, como el nombre del autor, descripción, palabras claves...
  • http-equiv="". Aquí se pueden poner el contenido de seguridad del sitio web.
  • content="". Especifica lo que se pone en el "name" o el "http-equiv"
  • name="viewport" content="width=device-width, initial-scale=1.0". Este atributo hace que la web se adapte al dispositivo. Es un uso específico y común del "name" y "content".

Título de Pestaña

Para ponerle un título a la pestaña actual, dentro del head, ponemos la etiqueta "title" e indicamos el título que queremos. Por ejemplo, en la pestaña actual podrá ver que se llama "Inicio"
