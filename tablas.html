Tabla Básica

Para hacer una tabla, lo primero que debes abrir la etiqueta "table". A partir de ahora, todo lo que se explique estará dentro de esta etiqueta

Para crear una nueva fila, usamos la etiqueta "tr", dentro de esa etiqueta, podemo crear tantas columnas como deseemos.

Para crear columna, tenemos dos etiquetas:

  • Etiqueta "th". Esta etiqueta tiene un formato de cabecera, con una letra en negrita.
  • Etiqueta "td". Esta etiqueta tiene un formato de texto normal, datos que se ponen en la tabla sin más.

Evidentemente se puede cambiar el formato de las dos etiquetas mediante CSS. A continuación tiene un ejemplo de una tabla básica:

Habrás podido observar que no te aparecen los bordes cuando haces la tabla, eso es porque debes añadir los siguientes valores en el CSS:

  1. "border: 'X'". Lo que sigue a "border" es el grosor del borde, medido en píxeles (px), o la medida que quieras usar
  2. "border-style: 'X'". Este es el tipo de borde, en el caso de querer el usual, es "solid".
  3. Para que haya una sola línea en vez de un cuadrado por etiqueta, debes usar la siguiente opción: "border-collapse: collapse;"
  4. Para cambiar el color, usa "border-color: 'color deseado'"

Colapso de Filas y Columnas

Si quieres juntas filas o columnas, usa los siguientes atributos dentro de la etiqueta que quieras:

  • "colspan="número"". Con esto (dentro de un th o td), haces que ocupe el número de columnas que quieres.
  • "rowspan="número"". Con esto (dentro de un th o td), haces que ocupe el número de filas que quieres.

Alternar Colores

Para colorear cada fila de un color, cuando hagas referencia a las filas (es decir, solo debes usar el "tr"), usa la pseudoclase "nth-child", y haz referencia, entre paréntesis, a las fila pares o impares (en ingés). Después aplicamos el "background-color", que es color de fondo que queremos.
