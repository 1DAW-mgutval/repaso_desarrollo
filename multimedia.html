MULTIMEDIA

Enlaces

Si quieres crear un enlace, usa la etiqueta "a", dentro de ella, el atributo "href=" y después pon el enlace donde quieres ir o incluso otro archivo html. En la etiqueta, pon lo que quieras que de clic la gente, ya sea un texto, o incluso una imagen como veremos.

Chiquito

Imágenes

Si quieres poner una imagen, usa la etiqueta "img", dentro usa el atributo "src=", donde haces referencia a un archivo en tu ordenador a una imagen de internet, mediante un enlace.

Para ajustar el tamaño de una imagen, usa el atributo "height=", para la altura, y "width=" para el ancho.



Audio

Para poner un archivo de audio, usa la etiqueta "audio", dentro usa la etiqueta "source" con los atributos: "src=" haz referencia al archivo, nuevamente enlace o archivo local, y "type=", indicando el tipo de archivo y la extensión, en mi caso sería "audio/mp3". Dentro de la etiqueta "audio", pon el atributo "controls", para que puedas ir para adelante, atrás, pausar...


Vídeo

Para poner un vídeo es igual que un audio, pero el nombre de la etiqueta es "video". Además, el "type=" de la etiqueta "source" cambia, con el tipo de archivo y extensión, en mi caso "video/mp4".
